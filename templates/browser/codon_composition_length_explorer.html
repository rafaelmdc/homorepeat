Overview layer

Codon preference and transition summaries

Preference: for 2-codon residues, each cell is a signed codon preference score. Positive values prefer the first codon, negative values prefer the second, and stronger color means stronger preference.

Dominance: for residues with 3 or more codons, each cell marks the dominant codon and uses strength to show how decisive that codon is.

Shift: each cell marks how much codon composition changes between adjacent length bins for the same taxon.

Similarity: pairwise trajectory divergence between visible taxa across all shared length bins. Lower values indicate more similar codon-length trajectories.
